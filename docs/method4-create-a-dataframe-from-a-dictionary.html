Method #4. Creating a DataFrame from a Dictionary

Python

import pandas as pd

data = {'col1': [1,2],
        'col2': [3,4],
        'col3': ['A','B']}

df = pd.DataFrame(data)

print(df)
   col1  col2 col3
0     1     3    A
1     2     4    B

R

# There is no dict in base R, the closest is probably list + vector
data = list(col1 = c(1,2), # col1 is a vector
            col2 = c(3,4),
            col3 = c('A','B'))

df = data.frame(data)

print(df)
  col1 col2 col3
1    1    3    A
2    2    4    B
